## Data Content Overview:
1. **`sales_records.csv`**: Contains 20 rows of sales data with intentional type traps, duplicate IDs, and multiple date formats.
2. **`product_catalog.html`**: A static HTML page with navigation bars, sidebars, and footers. The actual data is in a table.
3. **`legacy_pipeline.py`**: Old Python code containing business logic rules in docstrings and a misleading comment for the tax calculation.
4. **`demo_transcript.txt`**: A transcript of an AI lecture with noise tokens (`[Music]`, `[inaudible]`) and a price mentioned in Vietnamese words.
